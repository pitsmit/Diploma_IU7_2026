### Параметры журнала
* установить максимальный размер журнала;
* установить журналируемые события:
    * вставка;
    * извлечение;
    * чтение;
    * запись;
    * выполнение.

### Параметры теневого хранилища
* установить максимальный размер теневого хранилища;
* установить диапазон размера файлов, для которых будут создаваться теневые копии.

### Белый список устройств
* получить список;
* добавить устройство в список;
* удалить устройство из списка;
* установить дату окончания разрешения для устройства.

### Белый список процессов, имеющих право на работу с USB-накопителями
* получить список;
* добавить процесс в список;
* удалить процесс из списка;
* установить дату окончания разрешения для процесса.

### Просмотр журнала
* получить журнал;
* фильтр по периоду;
* фильтр по типу события;
* фильтр по VENDOR_ID/PRODUCT_ID/имени устройства.
